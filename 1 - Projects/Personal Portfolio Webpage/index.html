Technical Documentation Page

This project serves as a technical guide to a Simple Light Intensity Control System, designed to automatically adjust light brightness based on ambient conditions. The documentation includes sections covering system introduction, components, setup instructions, control system design using a Proportional (P) controller, and operational guidelines with troubleshooting tips. The page features a responsive layout with a navigation sidebar for easy access to different sections. Each section is detailed with clear headers, informative paragraphs, and structured lists, providing a comprehensive overview of both hardware (microcontroller, light sensor, LED) and software (control code) components.

Product Landing Page

This project showcases a responsive Product Landing Page for a Fitness Brand, designed to promote personalized training plans. The page features a dynamic header with the brand logo and navigation bar for seamless user experience.
